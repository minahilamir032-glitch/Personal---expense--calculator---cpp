# Personal Expense Calculator (C++)

## Description

Personal Expense Calculator is a beginner-level console application
developed in C++. This program allows users to enter their name, total
available money, and expenses, and then calculates the remaining
balance.

## Features

-   Simple and user-friendly console interface
-   Accurate expense calculation
-   Structured expense report
-   Beginner-friendly C++ project

## Concepts Used

-   C++ basic structure
-   Variables and data types
-   Input and output (cin, cout)
-   Arithmetic operators
